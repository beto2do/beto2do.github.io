About Buttons

Yare Buttons

This web component creates buttons easily. You have only to add a tag like this <yare-button value="click me"/> and set its attributes dynamically to render the custom button.

Button types

There are two kinds of buttons: <button> and <a> elements.

 <yare-button value="Button"/>
 <yare-anchor value="Anchor" href="https://themeforest.net/"/>


Colors

You can choose the button color changing the attr color="primary".

               
primary secondary success danger warning info light dark


Layouts

Change the button style setting a pattern. The default pattern is bootstrap buttons.

  pattern="outline"
  pattern="outline-circle"
  pattern="circle"


Sizes

size="sm"
  size="lg"


Icons

This component is supporting fontawesome icons. In order to render an icon you’ll need to add an attribute icon="xbox", but you have to remove fa- from fa-xbox.

  facebook-square
  google-plus
  amazon
  apple
  check-circle
